Pertama, yang ada awalan Update File: + posisi folder
yaitu ubah file sesuai posisi folder tersebut
ubah file nya dengan menghapus baris yang depannya di awali 1 karakter -
dan menambah baris yang depannya di awali 1 karakter +
tetapi tambahkan baris tanpa karakter +
Abaikan jika tidak di awali karakter - atau +
Kedua, yang ada awalan Delete File: + posisi folder
maka rename extention file tersebut menjadi .md
Ketiga, untuk awalan Add File: + posisi folder
maka tambahkan file tersebut sesuai posisi folder
sekalian tambahkan isinya sesuai instruksi
